In this article we will create a gallery to display all the notes in our application.

### Instruct
1. Go to **UI Library** in Studio. Select `NoteUI` then enter **Configure**
![](/images/5-Create-Collection-View/Create-Collection-View-1.png?featherlight=false&width=90pc)
2. Then select **Create Collection**.
![](/images/5-Create-Collection-View/Create-Collection-View-2.png?featherlight=false&width=90pc)
3. Then change the gallery style to grid and set the number of **columns 3** and set the **page size 9** which will eventually **add 10px** to each side of the note.
![](/images/5-Create-Collection-View/Create-Collection-View-3.png?featherlight=false&width=90pc)

You can set the sort or filter conditions for your collection on the right side.
