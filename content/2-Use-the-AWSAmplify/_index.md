In this lesson, We will automatically generate the original data in AWS Amplify data manager..

### Instruct

1. Go to the **Content** tab in Amplify Studio.
![](/images/2-Use-the-AWSAmplify-Data-Manager/Use-the-AWSAmplify-Data-Manager-1.png?featherlight=false&width=90pc)
2. In the **Content** page select **Actions** and select **Auto generate data**.
![](/images/2-Use-the-AWSAmplify-Data-Manager/Use-the-AWSAmplify-Data-Manager-2.png?featherlight=false&width=90pc)
Here you will be able to modify your application data. I will create 100 rows of data and add some constraints. I want my title to be 1-3 words long. I want each of my text fields to be 1-5 sentences long. Then I will click **Generate Data**.
![](/images/2-Use-the-AWSAmplify-Data-Manager/Use-the-AWSAmplify-Data-Manager-3.png?featherlight=false&width=90pc)
3. You should now see 100 rows of data in the Amplify data manager.
![](/images/2-Use-the-AWSAmplify-Data-Manager/Use-the-AWSAmplify-Data-Manager-4.png?featherlight=false&width=90pc)
You can access and edit any rows of data, create new notes, or delete them using the data manager.
