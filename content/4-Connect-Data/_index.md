In this lesson, we will link data generated by AWS Amplify to the UI Component.

### Instruct

1. Go to **UI Library** in Studio. Select `NoteUI` then enter **Configure**
![](/images/4-Connect-Data-Models/Connect-Data-Models-1.png?featherlight=false&width=90pc)
2. Click on the title on the existing card that says `T-Shirt`.
![](/images/4-Connect-Data-Models/Connect-Data-Models-2.png?featherlight=false&width=90pc)
On the right side of the page, Select **Set Text Label**. We'll want to make this the title of the note.

- Set the label holder: `note.title`.
![](/images/4-Connect-Data-Models/Connect-Data-Models-9.png?featherlight=false&width=90pc)
You should see the t-shirt text change to one of the titles of one of the generated notes. If you click **Shuffle Preview Data** you will cycle through different notes!
![](/images/4-Connect-Data-Models/Connect-Data-Models-3.png?featherlight=false&width=90pc)
3. Do the same for the note text - select the paragraph below the title.
- Set the label to `note.title`.
- Click on the trash can icon.
- Set `onClick` holder to `Delete` under data. Then select `Note` model and set ID to `note.id`
![](/images/4-Connect-Data-Models/Connect-Data-Models-4.png?featherlight=false&width=90pc)
#### Create a form

1. Back to **UI Library.**
2. Click on the `CreateNote` form, then select **Configure**. Click **Save** and add
![](/images/4-Connect-Data-Models/Connect-Data-Models-5.png?featherlight=false&width=90pc)
- Prop: `onClick.
- Action: `Create new`.
- Model: `Note`.
- Then select `title` input to fill in the title model field and the like for the text.
- The name `TextField` comes from Figma - you can rename the elements within it!
![](/images/4-Connect-Data-Models/Connect-Data-Models-6.png?featherlight=false&width=90pc)
#### Update form

1. Back to **UI Library.**
2. Click **UpdateNote** , then select **Configure**. Click **Save** and add as shown below.
![](/images/4-Connect-Data-Models/Connect-Data-Models-7.png?featherlight=false&width=90pc)
![](/images/4-Connect-Data-Models/Connect-Data-Models-8.png?featherlight=false&width=90pc)
We have now linked the data we created in Amplify Studio with the UI components we created in Figma.
